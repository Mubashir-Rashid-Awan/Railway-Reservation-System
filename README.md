# 🚆 Railway Reservation System

A console-based Railway Reservation System developed in C++ using Object-Oriented Programming (OOP), Data Structures, and File Handling concepts.

This project simulates a real-world railway booking system where users can view trains, book tickets, cancel reservations, manage waiting lists, and handle passenger records efficiently.

The system is designed using a modular multi-file architecture to improve code readability, maintainability, and scalability.



# 📌 Features

✅ Train Management System  
✅ Ticket Booking System  
✅ Ticket Cancellation System  
✅ Waiting List Management  
✅ Passenger Record Management  
✅ Admin Authentication Panel  
✅ File Handling for Data Storage  
✅ Doubly Linked List Implementation  
✅ Stack Implementation for Cancelled Tickets  
✅ Modular Multi-File OOP Structure  
✅ Console-Based User Interface
